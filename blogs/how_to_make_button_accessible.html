How to make a button accessible?

Before going how to make a button accessible? Please take look at my previous blog about accessibility.

I have chosen first to cover making button accessible because as the button is the main call to action for a website and we use more no of buttons in the website.

By using the button tag.

Button tag is an easy and simple way to make buttons accessible.While using the button tag the tab focus and role of a button are automatically assigned to a button. 

          <button>Help</button>

Making link as accessible button

To make the link accessible we need to add role='button' for the link, this will be helpful for screen reader users as it will convey as a button. 

          <a href='https://www.google.com/' target='_blank' role='button' class='btn'>Google</a>

Making icon as an accessible button.

Enclose the icon within the button tag and aria-label='label name' for the icon button. Since the icon button doesn't have a visual label. If we do not provide the label it will just say a button for the screen reader users. So we should provide the label for it. Similarly, you can make the image as a button. 

          <button class='btn' aria-label='Help' onclick='showalert()'><i class="fas fa-question-circle" ></i></button>

Making div as accessible button.

To make div as button we need add role as button to convey it as button . To give the keyboard tab focus we need to add tabindex as 0. 

          <div id='divButton' onclick='showalert()' tabindex='0' class='btn' role='button' >Help</div>

Apart from this, we need to add the following javascript code to access it using the keyboard enter or space key. 

          var divButton=document.querySelector('#divButton');

divButton.addEventListener("keydown", e => {
if (e.key === " " || e.key === "Enter" || e.key === "Spacebar")
{
divButton.click();
}
});

We have added the key down event listeners for the Spacebar and enter key and we made a click action using the javascript click function.

Take a look into the examples on codepen.

These are the ways how we can make the button accessible.
